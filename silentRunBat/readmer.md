# Run a bat file sirently

## usage

### vbs

`01_vbs/sirentRun.vbs`をダブルクリックして実行

一度もコンソールが出てくることなく, 最小化された状態で実行開始できる

### 2つのbatを使う

`02_double_bat/start.bat`をダブルクリックして実行

一瞬だけコンソールが出てきてすぐに最小化される

### 1つのbatだけで実行

`03_one_bat/start.bat`をダブルクリックして実行

一瞬だけコンソールが出てきてすぐに最小化される
